# SCM 분석 스킬

## 분석 기능 목록

### 1. ABC-XYZ 분석
- ABC: 매출 기여도 기준 분류 (A: 80%, B: 15%, C: 5%)
- XYZ: 수요 변동계수(CV) 기준 (X: <0.5, Y: 0.5~1.0, Z: >1.0)
- 매트릭스: AX~CZ 9가지 조합별 관리 전략 도출

### 2. 안전재고 계산
```
안전재고 = Z값 × √(리드타임 × 수요분산² + 평균수요² × 리드타임분산²)
```
- Z값: 서비스 수준에 따라 결정 (95% → 1.65, 99% → 2.33)

### 3. 발주점 (Reorder Point)
```
발주점 = 일평균판매량 × 리드타임(일) + 안전재고
```
- 프로젝트 전체에서 단일 공식만 사용할 것

### 4. 재고일수 (Days of Inventory)
```
재고일수 = 현재고 / 일평균판매량
```
- 일평균판매량 = 0이면 ∞로 표시 (0 나눗셈 방지)

### 5. 재고상태 분류 (단일 기준으로 통일)
| 상태 | 조건 |
|------|------|
| 부족 | 현재고 < 안전재고 |
| 주의 | 안전재고 ≤ 현재고 < 안전재고 × 1.5 |
| 적정 | 안전재고 × 1.5 ≤ 현재고 < 안전재고 × 3.0 |
| 과다 | 현재고 ≥ 안전재고 × 3.0 |

### 6. 수요예측
- 가중이동평균 (최근 데이터 가중)
- 지수평활법 (알파 자동 최적화)
- 계절성 인덱스 적용

## 규칙
- 계산 결과는 소수점 2자리까지
- 0 나눗셈 항상 방지
- 매직 넘버 사용 금지 (상수로 정의)
- 단위 명시 (원, 개, 일, %)
- 한국어 용어 사용
