## 1. 디자인 철학

### 1.1 핵심 원칙: "3초 안에 파악, 1클릭에 액션"

경쟁사들의 **"90년대 스타일 UI"** 문제를 정면으로 공략합니다.

| 원칙 | 설명 | 경쟁사 대비 |
|------|------|------------|
| **Glanceable** | 3초 안에 현재 상태 파악 | vs 정보 과부하 |
| **Actionable** | 다음 행동이 명확함 | vs 액션 불명확 |
| **Forgiving** | 실수해도 쉽게 복구 | vs 복잡한 되돌리기 |
| **Progressive** | 필요한 정보만 점진적 노출 | vs 모든 것을 한꺼번에 |
| **Consistent** | 예측 가능한 인터랙션 | vs 일관성 없는 UI |

### 1.2 브랜드 포지셔닝

> **"공급망의 노션(Notion)"** - SCM 전문가를 위한 AI 기반 올인원 플랫폼

- **설정 30분** (경쟁사: 수일~수주)
- **학습 곡선 제로** (경쟁사: 가파른 학습 곡선)
- **한눈에 파악** (경쟁사: 5단계 메뉴 탐색)

### 3.2 재고상태 7단계 컬러 (핵심 차별화)

SCM 도메인의 핵심인 재고상태를 **색상만으로 즉각 인지**할 수 있도록 설계합니다.

| 상태 | 조건 | HEX | Tailwind | 의미 |
|------|------|-----|----------|------|
| **품절** | 현재고 = 0 | `#18181b` | `zinc-900` | 즉시 대응 필요 |
| **위험** | 0 < 현재고 < 안전재고 × 0.5 | `#ef4444` | `red-500` | 긴급 발주 |
| **부족** | 안전재고 × 0.5 ≤ 현재고 < 안전재고 | `#f97316` | `orange-500` | 발주 권장 |
| **주의** | 안전재고 ≤ 현재고 < 발주점 | `#eab308` | `yellow-500` | 모니터링 |
| **적정** | 발주점 ≤ 현재고 < 안전재고 × 3.0 | `#22c55e` | `green-500` | 최적 상태 |
| **과다** | 안전재고 × 3.0 ≤ 현재고 < 안전재고 × 5 | `#3b82f6` | `blue-500` | 판매 촉진 검토 |
| **과잉** | 현재고 ≥ 안전재고 × 5.0 | `#a855f7` | `purple-500` | 재고 처분 검토 |

## 4. 타이포그래피

### 4.1 폰트 패밀리

```css
/* 한글 최적화 */
--font-sans: 'Pretendard', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* 숫자/코드 */
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
```

### 4.2 타입 스케일

| 용도 | 크기 | 무게 | Line Height | Tailwind |
|------|------|------|-------------|----------|
| **H1** | 30px | 700 | 1.2 | `text-3xl font-bold` |
| **H2** | 24px | 600 | 1.3 | `text-2xl font-semibold` |
| **H3** | 20px | 600 | 1.4 | `text-xl font-semibold` |
| **H4** | 16px | 600 | 1.4 | `text-base font-semibold` |
| **Body** | 14px | 400 | 1.5 | `text-sm` |
| **Body Large** | 16px | 400 | 1.5 | `text-base` |
| **Caption** | 12px | 400 | 1.4 | `text-xs` |
| **Label** | 14px | 500 | 1.4 | `text-sm font-medium` |

### 4.3 타이포그래피 원칙

1. **본문 최소 14px** - 가독성 우선 (경쟁사: 12px 이하)
2. **한글 행간 1.5** - 한글 특성 고려
3. **숫자는 모노스페이스** - 정렬 및 가독성
4. **명확한 계층** - H1 > H2 > H3 > Body > Caption

## 10. 접근성 (A11y)

### 10.1 색상 대비

- **텍스트**: 최소 4.5:1 대비율 (WCAG AA)
- **큰 텍스트 (18px+)**: 최소 3:1 대비율
- **인터랙티브 요소**: 최소 3:1 대비율

### 10.2 키보드 네비게이션

- **Tab**: 순차 이동
- **Enter/Space**: 활성화
- **Escape**: 모달/드롭다운 닫기
- **Cmd+K / Ctrl+K**: 검색 열기
- **Arrow Keys**: 메뉴 내 이동

### 10.3 포커스 표시

```css
/* 포커스 링 */
:focus-visible {
  outline: 2px solid var(--primary-500);
  outline-offset: 2px;
}
```
