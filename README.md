# 스마트 발주 시스템

재고 관리 및 자동 발주 시스템

## 주요 기능

- 📊 실시간 재고 현황 대시보드
- 🔔 스마트 발주 알림 (재고 부족 자동 감지)
- 📈 ABC-XYZ 분석 기반 재고 최적화
- 🎯 안전재고 및 발주점 자동 계산
- 📦 추천 발주 물량 자동 산출

## 기술 스택

- **Frontend**: Streamlit
- **Data Analysis**: Pandas, NumPy
- **Visualization**: Plotly
- **Data Source**: Excel (OpenPyXL)

## 배포 URL

https://smart-procurement.streamlit.app

## 로컬 실행

```bash
pip install -r requirements.txt
streamlit run smart_procurement.py
```

## 데이터 파일
