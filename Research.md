## Part 3: 사용자 Pain Points

### 3.1 Top 10 불만사항

| 순위 | 불만 | 영향도 |
|------|------|--------|
| 1 | **실시간 데이터 동기화 안됨** | 과잉판매, 품절 직결 |
| 2 | **시스템 통합 어려움** | WMS 구현 60% 지연/초과 |
| 3 | **숨겨진 비용** | 온보딩, 교육, 마이그레이션 |
| 4 | **수요 예측 부정확** | 재고 왜곡 비용 $8,180억/년 |
| 5 | **수동 작업** | 중소기업 43% 수동 추적, 직원 시간 40% 소비 |
| 6 | **학습 곡선** | 모든 주요 솔루션 공통 |
| 7 | **고객 지원 부실** | "설정만 해주고 떠남" |
| 8 | **확장성 한계** | 성수기 대량 처리 불가 |
| 9 | **데이터 품질** | 오래된 마스터 → 오피킹 |
| 10 | **직원 변화 저항** | 디지털 전환 실패의 67% |

### 3.2 기존 솔루션에 없는 것 (사용자가 원하지만 없는 기능)

1. **AI 자동 발주 제안** (저렴한 가격에)
2. **공급업체 실시간 협업 포털**
3. **산업별 특화** (식품 유통기한, 패션 사이즈/시즌)
4. **지속가능성 추적** (탄소 발자국, 폐기물)
5. **커뮤니티 기반 벤치마킹** (동종업계 비교)
6. **모바일 우선 경험**
7. **직관적 UI** (현재 솔루션 대부분 90년대 스타일)

### 3.3 핵심 재무 수치

| 지표 | 수치 |
|------|------|
| 재고 관리 미흡 시 매출 손실 | 연간 매출의 **최대 11%** |
| 운전 자본 묶임 | 재고 가치의 **25-35%** |
| 긴급 배송 비용 | 정상 대비 **400-600%** |
| 수동 재고 작업 시간 | 창고 직원 시간의 **40%** |

## Part 4: 경쟁 공백 분석 (White Space)

### 4.1 발견된 공백

| 공백 유형 | 상세 | 기회 크기 |
|-----------|------|-----------|
| **가격 공백** | 월 $10-50 구간 SCM 도구 부재 | ★★★★★ |
| **기능 공백** | "AI + 커뮤니티 + SCM" 결합 제품 **전무** | ★★★★★ |
| **한국어 공백** | 한국어 지원 SCM SaaS 심각하게 부족 | ★★★★☆ |
| **커뮤니티 공백** | SCM 전문가 커뮤니티 사실상 없음 | ★★★★★ |
| **교육 공백** | 온라인 SCM 교육 접근성 낮음 | ★★★☆☆ |
| **산업 특화 공백** | 식품, 패션, 화학 등 버티컬 SCM 부족 | ★★★★☆ |

### 4.2 블루오션 확인

> **"SCM + 커뮤니티 + AI" 결합 제품은 현재 시장에 존재하지 않는다.**

- AI 기반 SCM은 있다 → 전부 엔터프라이즈급 ($10K+/월)
- SCM 소프트웨어는 있다 → 커뮤니티 기능 전무
- 직업 커뮤니티는 있다 → SCM 전문 커뮤니티 없음 (한국: 카톡 1,700명이 전부)
- **세 가지를 합친 제품 = 블루오션**

### 5.2 SCM 직업군 규모

- 잡코리아 SCM 채용 공고: **~1,018건** (상시)
- 물류관리사 응시: 2024년 **7,186명** (증가 추세)
- APICS CPIM 보유자: 누적 **~2,000명**

### 5.3 기존 커뮤니티의 한계 = 우리의 기회

1. **대규모 온라인 커뮤니티 부재** → 우리가 만든다
2. **실무 노하우 공유 플랫폼 없음** → 우리가 만든다
3. **전문가 네트워킹 채널 부재** → 우리가 만든다
4. **온라인 교육 접근성 낮음** → AI 튜터로 해결
5. **SCM 전문 콘텐츠 없음** → 우리가 만든다

## Part 7: GTM(Go-to-Market) 전략

### 7.1 가격 전략

**권장 가격 구조**:
| 플랜 | 가격 | 대상 | 핵심 기능 |
|------|------|------|-----------|
| **Free** | $0 | 개인/마이크로 | 10 SKU, 기본 재고관리, 커뮤니티 접근 |
| **Starter** | $29/월 | 소기업 | 100 SKU, AI 예측 기본, 보고서 |
| **Pro** | $99/월 | 성장기업 | 1,000 SKU, AI 전체, 자동화, 팀 |
| **Enterprise** | 커스텀 | 중견기업 | 무제한, 전용 지원, API |

- Freemium 전환율 목표: 5-10%
- 한국 시장: 경쟁사 대비 20-30% 저렴하게 시작

### 7.2 PLG (Product-Led Growth) 전략

- **순수 PLG**: SCM의 복잡성 때문에 어려움
- **추천**: Product-Led Sales (하이브리드)
  - SMB → Self-serve (무료 도구 → 유료 전환)
  - 중견기업 → PLG + 세일즈
  - 대기업 → Sales-led

### 7.3 Cold Start 해결 (0→100 고객)

| 단계 | 기간 | 전략 | 목표 |
|------|------|------|------|
| 0→10 | 3-6개월 | 네트워크 활용, 창업자 세일즈 | 인사이트 수집 |
| 10→100 | 6-12개월 | 무료 도구, 콘텐츠 마케팅, 리퍼럴 | 반복 패턴 발견 |
| 100→1K | 12-24개월 | 커뮤니티, 세그먼트별 전략 | 확장 시스템 |

**무료 도구로 유입 유도**:
- Excel 재고관리 템플릿 (무료 배포)
- PSI 계산기 (웹 도구)
- ABC 분석 도구 (무료)
- 발주점 계산기 (무료)
- → 이 도구 사용자를 풀 제품으로 전환

### 7.4 한국 B2B 시장 특수성

- 구매 의사결정에 평균 **5.4명** 승인 필요
- **레퍼런스가 절대적** ("누가 쓰나?"가 "무엇을 하나?"보다 중요)
- 수치화된 **ROI 제시 필수**
- **대면 영업**이 여전히 중요

## Part 8: 전략적 포지셔닝

### 8.1 최종 포지셔닝

> **"공급망의 노션(Notion)" - SCM 전문가를 위한 AI 기반 올인원 플랫폼**

**핵심 가치 제안**:
- 설정 **30분** (경쟁사: 수일~수주)
- 월 **$29**부터 (경쟁사: $100-500+)
- **AI 자동 예측/발주** (경쟁사: 수동 또는 엔터프라이즈급만)
- **커뮤니티 벤치마킹** (경쟁사: 아무도 없음)
- **한국어 완벽 지원** (경쟁사: 거의 없음)

### 8.2 왜 이 포지셔닝인가?

| 비교 축 | 기존 시장 | 우리 |
|---------|----------|------|
| 가격 | $100-500/월 | **$0-29/월로 시작** |
| 설정 시간 | 수일~수주 | **30분** |
| AI 기능 | 엔터프라이즈 전용 | **모든 플랜에 포함** |
| 커뮤니티 | 없음 | **핵심 차별화** |
| 한국어 | 미지원 또는 부실 | **완벽 지원** |
| 학습 곡선 | 가파름 | **AI 튜터로 해결** |

### 8.3 경쟁 우위(Moat) 구축

| Moat 유형 | 전략 | 효과 시점 |
|-----------|------|-----------|
| **데이터 네트워크** | 더 많은 기업 → 더 나은 AI 예측 | 24개월 후 추격 불가 |
| **커뮤니티** | 벤치마킹, Q&A, 템플릿 공유 | 12개월 후 이탈 방지 |
| **전환 비용** | 2-3년 축적 데이터 + 워크플로우 | 자연스러운 Lock-in |
| **AI 학습 데이터** | 독점 산업별 공급망 데이터셋 | 36개월 후 복제 불가 |

### 8.4 타겟 세그먼트 우선순위

## Part 10: 핵심 결론

### 우리가 이길 수 있는 이유 5가지

1. **블루오션 확인됨**: SCM + AI + 커뮤니티 결합 제품이 세상에 없다
2. **한국 SCM 커뮤니티가 비어있다**: 카톡 1,700명이 전부. 선점하면 메디게이트처럼 될 수 있다
3. **가격 공백이 크다**: 월 $10-50 구간 SCM 도구 부재. 이카운트(4만원)과 Cin7($349) 사이가 텅 비어있다
4. **기존 솔루션 UI가 구리다**: "90년대 스타일 UI" 불만이 Fishbowl, inFlow, Oracle 전부에 있다
5. **AI-Native 장점**: 레거시 없이 처음부터 AI 중심으로 설계 가능

### 우리의 리스크 3가지

1. **Cold Start**: 첫 100명이 가장 어렵다 → 무료 도구 + 콘텐츠로 해결
2. **Python→TypeScript 포팅**: 로직 오류 리스크 → Phase A 테스트로 완화
3. **1-2명 팀의 한계**: 범위 관리 필수 → MVP 범위 엄격히 제한

### 한 줄 요약

> **한국 중소 제조업을 위한 AI 기반 재고관리 SaaS + SCM 전문가 커뮤니티.**
> **"설정 30분, 월 29달러, AI가 알아서 발주해주고, 같은 업계 사람들과 노하우를 나눌 수 있는 곳."**
